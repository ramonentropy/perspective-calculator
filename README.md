# Perspective Calculator v1.2a

**Entropy Studio** · VFX Production Tool

A browser-based tool for matching the perspective of a subject between two different shots or locations, maintaining the geometric relationship between camera, actor and framing.

## What it does

Enter the data from the original shot in **Plate 01** and the conditions of the new setup in **Plate 02**. The tool automatically calculates the values needed to reproduce the perspective.

When an exact match is not possible, it also allows you to modify distances, heights, cameras or focal lengths to find the best alternative solution.

## Features

- Geometric perspective matching between two camera setups
- 3D viewer with Graf and Three.js model (Mix / Split / Wipe comparison)
- Advanced mode with real camera database (ARRI, Sony, RED, Canon…)
- Side View diagram (Distance and Tilt modes)
- JSON export / import for session archiving
- PDF report generation
- Dark / light mode · Desktop / mobile layout

## Usage

Open `index.html` directly in a browser — no installation required.
